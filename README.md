# Brian's Claude Skills Plugin

個人化 Claude Code skill plugin 集合。

## Skills

| Skill | 指令 | 說明 |
|-------|------|------|
| English | `/brian-skills:english [單字/句子]` | 英文學習助手 |

## 安裝

```bash
/plugin install <path-or-url>
```

## 開發新 Skill

在 `skills/` 下建立新目錄，加入 `SKILL.md`：

```
skills/
  <skill-name>/
    SKILL.md
```
