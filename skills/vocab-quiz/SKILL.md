使用者英文程度為 B2。

從當前目錄的 `vocabulary.json` 讀取單字表，進行抽考。

## 流程

1. 用 Bash 讀取 `vocabulary.json`
2. 若檔案不存在或單字表為空，告知使用者先用 `/english-learn` 記錄單字
3. 根據 `$ARGUMENTS` 決定出題數量與題型（見下方規則）
4. 隨機抽取單字出題
5. 等待使用者作答，然後批改並給解析

## 參數規則

- 無參數：隨機出 5 題，混合題型
- 數字（例如 `10`）：出指定題數，混合題型
- 題型關鍵字：
  - `中翻英` → 全部出中翻英
  - `英翻中` → 全部出英翻中
  - `造句` → 全部出造句題
  - `填空` → 全部出填空題
- 可組合，例如 `10 中翻英`

## 題型格式

**中翻英：**
```
Q1. 「有韌性的」的英文是？
```

**英翻中：**
```
Q1. "resilient" 的繁中意思是？
```

**造句：**
```
Q1. 請用 "resilient" 造一個英文句子
```

**填空：**
（用該單字的例句，將單字替換為空格）
```
Q1. 填入正確的單字：
She remained ________ despite the setbacks.
```

## 出題格式

一次列出所有題目，等使用者全部作答後再一起批改。

```
📝 今日單字測驗（5題）
━━━━━━━━━━━━━━━━━━
Q1. ...
Q2. ...
Q3. ...
Q4. ...
Q5. ...
━━━━━━━━━━━━━━━━━━
請作答後貼上，我來批改。
```

## 批改格式

```
📊 批改結果：4/5 答對

✅ Q1. resilient → 正確！
❌ Q2. 「堅持」→ 你答：insist，正確答案：persist
   persist = 堅持不懈地做某事（通常面對困難）
   insist = 堅持某個主張或要求（語意不同）
✅ Q3. ...

💡 建議複習：persist, ...
```

批改時針對 B2 程度給解析，重點說明容易混淆的地方。
